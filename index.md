# Kimi For Coding 会员权益

Kimi For Coding 是 Kimi 会员计划中面向代码开发场景提供的增值会员权益。

* 兼容多个场景：兼容 Kimi CLI、Claude Code、RooCode，轻松融入各类开发流程
* 更快更稳响应：每 5 小时 100-500 次高速请求，全天高速响应，无需等待

> 未订阅用户前往 [Kimi](https://www.kimi.com/) 登录后点击左下角用户头像，进入“会员计划”即可订阅体验

> 已订阅的用户可在会员页弹窗获取 API Key 并查询使用额度

## 开始使用 Kimi For Coding 会员权益

### 第一步：查询您的 Kimi For Coding API Key
点击左下角用户头像 - 设置 - 管理 - 订阅，进入如下页面

![在“订阅”页的 Kimi For Coding 蓝字](screenshots/index/subscription.jpg)

点击 Kimi For Coding 蓝字，唤起弹窗，在弹窗中复制并保存您的 API Key

### 第二步：配置 API Key 至您的开发工具

* [在 Kimi CLI 中配置 Kimi For Coding 会员权益](./kimi-cli.html)
* [在 Claude Code 或 Roo Code 中配置 Kimi For Coding 会员权益](./third-party-agents.html)
