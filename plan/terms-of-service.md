# 使用条款

Kimi Coding 包月计划适用于使用 KimiCLI 的场合。我们深知 KimiCLI 仍有较大的优化空间，因此，为了更好地满足开发者预期，我们扩充了 KimiCoding 包月计划的使用范围：开发者们也可以在 Claude Code 和 Roo Code 中使用 Kimi Coding 包月计划提供的 API Key，未来还会支持更多 Coding Agent。



请注意，Kimi Coding 包月计划仅适用于 KimiCLI、Claude Code 和 Roo Code 产品，在非上述产品中使用 Kimi Coding 包月计划对应的 base\_url 和 API Key 均有可能被我们视为滥用，并可能导致会员权益停用或账号封禁。



**Kimi coding：** 包月订阅模式，用于个人开发应用，不要用于企业开发场景，适用于在 Kimi CLI 和其他 Coding Agent 中进行编程相关场景使用，非编程场景或超出限定范围的使用，可能会被判定为违规，导致会员权益暂停。

**Kimi 开放平台：** 预充值按量付费模式，可指定模型，满足更广泛的模型使用场景和产品集成。
