# 计费方式

## 计费逻辑

Kimi CLI 作为 Kimi 包月会员权益之一，其费用包含在套餐费用中；即 Kimi 会员使用**在会员页弹窗上生成的** API Key 进行编程时，仅统计请求次数并刷新可用额度，用户无需额外付费。

不同会员档位的请求额度（由于存在营销活动，实际价格和额度以会员页展示为准）：

* 49 元 Andante 套餐：每周 1024 次请求

* 99 元 Moderato 套餐：每周 2048 次请求

* 199 元 Allegretto 套餐：每周 7168 次请求


## 额度刷新逻辑

以用户订阅日期 T1 为起点，请求额度刷新方式为：

* 订阅生效后即获得首周额度，额度有效期为 T1 ～ T7

* T8 额度刷新，额度有效期为 T8 ～ T14

* 以此类推

如需变更套餐或取消订阅，权益变动情况以 Kimi用户服务协议 为准，可点击跳转查看详情。
