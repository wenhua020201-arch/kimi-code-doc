# Kimi Code Usage Policy

Kimi Code is dedicated to providing intelligent and efficient AI programming services for developers worldwide. Through continuous model capability advancement, relentless refinement of Agent product features, and the innovative Token Plan paradigm, we aim to shorten the path from imagination to reality, making general-purpose Agents accessible to every individual with an idea. To ensure a better experience for every Kimi Code user, we establish the following usage policy.


## Usage Boundaries

Kimi Code subscriptions are for interactive programming tool use only (Kimi CLI, VS Code, Claude Code, OpenCode, OpenClaw, etc.). The API Key included in your plan must not be used for self-built applications, SaaS products, server-side proxies, or any non-local coding tool scenarios. For such use cases or enterprise-level integration, please visit the [Kimi Platform](https://www.moonshot.ai/) or contact [code@moonshot.ai](mailto:code@moonshot.ai), and our sales team will be happy to assist you.


## 🚫 Platform Red Lines

To ensure stable model services for all Kimi Code members, the following behaviors are prohibited:

- Reselling accounts, selling API Keys for commercial purposes, or providing access to third parties without permission
- Aggregating, forwarding, or redistributing Kimi Code inference capabilities in any form
- Falsifying or tampering with client identity information
- Engaging in model distillation training or similar activities

Violations will result in rate limiting or account suspension, depending on the severity.

:::info 🤖 Appeal a False Positive
If you believe you have been incorrectly rate-limited or banned, please submit an appeal through the [Kimi website](https://www.kimi.com/) by clicking your avatar → User Feedback. Include your account ID, ban time, usage scenario description, and supporting screenshots. If verified as a false positive, we will lift the restriction immediately and offer additional quota as an apology for the inconvenience.
:::


## ✨ We Welcome

Kimi Code welcomes every creator who changes the world with code—whether you are an independent developer, an open-source contributor, or a beginner turning your first idea into a product. We especially encourage sustained and focused usage. For users who maintain long sessions and fully reuse historical context, the platform dynamically optimizes quota efficiency, making your Tokens go further for the same amount of work.

If you'd like to connect with fellow Kimi Coders and exchange technical insights, feel free to join our Feishu user group. Happy Coding!

## 💬 Frequently Asked Questions

**Q: I am a heavy user and use Kimi Code intensively every day. Will I be mistakenly banned?**

**A:** No. Risk control takes multiple signals into account. Heavy users with normal IP addresses, conversation states, and other usage indicators are typically not subject to enforcement actions.

**Q: Will I be mistakenly banned if I access Kimi through a VPN or company proxy?**

**A:** Accessing Kimi through a proxy (forward proxy) is fine. Sharing your account with others for them to access via proxy (reverse proxy) is a violation.
