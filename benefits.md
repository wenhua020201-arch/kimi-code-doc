# 权益说明

## 计费方式

### 计费逻辑

Kimi For Coding 作为 Kimi 包月会员权益之一，其费用包含在套餐费用中；即 Kimi 会员使用**在会员页弹窗上生成的** API Key 进行编程时，仅统计请求次数并刷新可用额度，用户无需额外付费。

不同会员档位的请求额度（由于存在营销活动，实际价格和额度以会员页展示为准）：

* 49 元 Andante 套餐：每周 1024 次请求
* 99 元 Moderato 套餐：每周 2048 次请求
* 199 元 Allegretto 套餐：每周 7168 次请求

### 额度刷新逻辑

以用户订阅日期 T1 为起点，请求额度刷新方式为：

* 订阅生效后即获得首周额度，额度有效期为 T1 ～ T7
* T8 额度刷新，额度有效期为 T8 ～ T14
* 以此类推

如需变更套餐或取消订阅，权益变动情况以 [《Kimi 付费服务协议》](https://www.kimi.com/user/agreement/paidServiceAgreement?version=v2) 为准。

## 使用条款

Kimi For Coding 会员权益适用于使用 Kimi CLI 等软件的场合。我们深知 Kimi CLI 仍有较大的优化空间，因此，为了更好地满足开发者预期，我们扩充了 Kimi For Coding 会员权益的使用范围：开发者们也可以在 Claude Code 和 Roo Code 中使用 Kimi For Coding 会员权益提供的 API Key，未来还会支持更多 Coding Agent。

请注意，Kimi For Coding 会员权益适用于 Kimi CLI，但不禁止用户自行在 Claude Code 和 Roo Code 产品中使用，在非上述产品中使用 Kimi For Coding 会员权益对应的 base\_url 和 API Key 均有可能被我们视为滥用，并可能导致会员权益停用或账号封禁。

**Kimi For Coding**： 会员权益，用于个人开发应用，不要用于企业开发场景，适用于在 Kimi CLI 和其他 Coding Agent 中进行编程相关场景使用，非编程场景或超出限定范围的使用，可能会被判定为违规，导致会员权益暂停。

**Kimi 开放平台：** 预充值按量付费模式，可指定模型，满足更广泛的模型使用场景和产品集成。
