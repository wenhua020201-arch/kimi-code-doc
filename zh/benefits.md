# 权益说明

Kimi Code 是 Kimi 会员权益中的一项服务，为会员提供编程相关的支持和功能，帮助会员在个人开发的过程中更高效地完成编程任务。

**请注意，本权益仅用于个人开发非用于企业开发场景，同时，用户应在 Kimi CLI 和其他我们允许的 Coding Agent 中进行编程相关场景使用。企业开发需求请关注 Moonshot 开放平台，开放平台支持预充值按量付费模式，可指定模型，满足更广泛的模型使用场景和产品集成。**

## 计费方式

### 计费逻辑

KimiCode额度作为 Kimi 会员权益之一，其费用包含在套餐费用中，用户无需额外付费, 可以通过以下途径使用：

- Kimi订阅会员使用在[Kimi Code控制台](https://www.kimi.com/code/console)生成的 API Key 在我们允许的平台进行编程时，仅消耗套餐内的请求次数并刷新可用额度。

- 在Kimi Code for CLI或者Kimi Code for VScode中使用Kimi账号Login后，也会消耗对应套餐内的Kimi Code额度

**不同会员档位的请求额度以会员展示页为准。**


### 额度刷新逻辑

以用户订阅日期 D1 为起点，请求额度刷新方式为：

- 订阅生效后立即获得首周额度，额度有效期为 D1 ～ D7
- D8 额度刷新，额度有效期为 D8 ～ D14
- 以此类推

## 使用范围

Kimi Code 权益原则上适用于使用 Kimi CLI 的场合。我们深知 Kimi CLI 仍有较大的优化空间，因此，为了更好地满足开发者预期，开发者们也可以自行在 Claude Code 和 Roo Code 中配置和使用 Kimi Code 权益提供的 API Key，在上述平台使用时请求次数也会被消耗，我们未来还会支持更多 Coding Agent。

**请注意，我们不禁止用户自行在 Claude Code 和 Roo Code 产品中使用上述权益，但在非上述产品中使用 Kimi Code 权益对应的 base_url 和 API Key 有可能被我们视为滥用和违规，导致会员权益停用或账号封禁。**

如需变更套餐或取消订阅，请参考 [《Kimi 付费服务协议》](https://www.kimi.com/user/agreement/paidServiceAgreement?version=v2) 和相关页面。
