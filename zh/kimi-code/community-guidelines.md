# Kimi Code 社区倡议

Kimi Code 致力于为全球用户提供智能、高效的 AI 编程服务，用模型能力的持续跃迁、Agent产品力的极致打磨、Token Plan的范式革新，缩短人类想象到现实的路径，让通用agent普惠每个有想法的个体。为保证每位Kimi Code用户拥有更好的体验，我们提出以下倡议。

## 适用范围

Kimi Code 订阅仅限编程工具交互式使用（Kimi CLI、VS Code 、Claude Code、OpenCode、OpenClaw 等），作为开源生态的共建者，如果您开发的工具或框架希望接入 Kimi Code 能力，欢迎直接联系 [code@moonshot.ai](mailto:code@moonshot.ai)。如有企业级集成、商业化服务或其他开放平台相关需求，也可前往 [Kimi 开放平台](https://platform.moonshot.cn) 了解更多合作方案。

为了让每个人都能持续获得稳定的模型服务，有一些使用规范我们需要大家一起遵守。

## 常见问题

**Q1: 我是重度用户，每天高强度使用，会被误封吗？**

不会。风控识别会考虑多种特征，对于 IP 地址、对话状态等使用指标正常的重度用户通常不会触发处置。

**Q2：我用 VPN 或公司代理访问，会被误封吗？**

通过代理访问 Kimi（正向代理）没问题；把自己的账号给别人转发（反向代理），属于违规。

**Q3: 我同时在多个设备或工具上使用 Kimi Code，会被误封吗？**

不会。开发者在不同设备（如工作电脑、个人笔记本）或使用不同编程工具（Kimi CLI、VS Code、Claude Code）之间切换，属于完全正常的使用模式。只要不刻意篡改客户端身份信息就不会被误封。
